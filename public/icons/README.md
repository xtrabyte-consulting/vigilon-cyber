# Vigilon Cyber Icons

This directory contains SVG icons used throughout the Vigilon Cyber website.

## Icon List

- **shield.svg** - Security, protection, compliance
- **lock.svg** - Encryption, security, authentication
- **certificate.svg** - Certifications, compliance documents, standards
- **check-circle.svg** - Verification, approval, completed tasks
- **document.svg** - Documentation, reports, files
- **chart.svg** - Analytics, assessments, data
- **cloud.svg** - Cloud security, cloud services
- **network.svg** - Connectivity, IoT, network security
- **alert.svg** - Warnings, urgent notifications, risks
- **search.svg** - Penetration testing, discovery, scanning
- **target.svg** - Threat targeting, vulnerability assessment
- **code.svg** - Software, development, testing
- **users.svg** - Teams, consulting, support
- **eye.svg** - Monitoring, visibility, auditing

## Usage

In Astro components:
```astro
<img src="/icons/shield.svg" alt="Security" class="icon-lg" />
```

With color (icons use currentColor):
```html
<img src="/icons/shield.svg" alt="Security" style="color: var(--color-accent)" />
```

## Color Guidelines

- Use Electric Cyan (#00D4FF) for interactive/tech features
- Use Intelligent Blue (#1E5F8C) for secondary elements
- Use Midnight Blue (#0B1F3F) for primary elements
- Use Alert Orange (#FF6B35) for warnings/CTAs
